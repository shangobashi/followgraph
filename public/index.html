FollowGraph

A privacy-first Chrome and Edge extension that scans your X/Twitter Following page and exports your following list without sending data to any server.

View on GitHub Install Instructions

Why it works

X uses a virtualized list, so off-screen users disappear from the DOM. FollowGraph extracts users during the scroll loop instead of waiting until the end.

What it exports

Unique accounts, profile URLs, last-seen activity timestamps when available, and CSV or JSON downloads.

Privacy model

No servers, no OAuth, no tracking. Data stays in the browser and the latest scan is stored locally.

Quick start

  1. Clone the repo and run npm install.
  2. Build the extension with npm run build.
  3. Open chrome://extensions or the Edge extensions page.
  4. Enable developer mode and load the generated extension/ folder.
